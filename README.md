# 🏭 Industrial Automation Using Siemens PLC

A comprehensive industrial automation project demonstrating PLC programming, Ladder Logic development, Siemens TIA Portal, Human Machine Interface (HMI), and SCADA integration for process control and monitoring.

---

## 📖 Project Overview

This project demonstrates the implementation of industrial automation concepts using Siemens PLCs. It covers PLC programming, Ladder Logic design, HMI development, SCADA integration, and process simulation using Siemens TIA Portal.

A practical case study involving a PLC-controlled Surface Grinding Machine is included to demonstrate industrial control logic and process automation.

## 🏗 Project Case Study

### PLC-Controlled Surface Grinding Machine

The project implements an automated control sequence for a surface grinding machine.

### Operation Sequence

1. Start button initiates the grinding motors.
2. Magnetic coil is activated after a predefined delay.
3. Conveyor moves forward at full speed.
4. Sensor inputs regulate conveyor speed.
5. Conveyor reverses after end detection.
6. Home sensor resets the system to its initial state.
